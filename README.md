Overview
This project is an AI-powered calling agent that can make phone calls and deliver information using voice synthesis.

Features
- Converts text to speech using ElevenLabs
- Uses MCP server for context handling
- Makes automated calls using Twilio
- Can respond to user commands (e.g., call a contact and give updates)

Tech Stack
- ElevenLabs
- MCP Server (Model Context Protocol)
- Twilio
- Cursor IDE

How it works
1. User gives command (e.g., "Call my friend")
2. MCP processes context
3. ElevenLabs generates voice
4. Twilio places the call
